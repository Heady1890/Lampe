
This is a C program for a self build lamp with following features:
-it can handle four different kinds of LED's (red, green, blue, white)
-it generates cpu efficient PWM signals
-it can display ever color with the mixture of these LED's
-it uses the HSV color model
-it can display several animations of colors in an endless loop

for the future:
-it will communicate over CAN
-it will measure the temperature of the LED's and fit the rotations of the fan.
-it will be possible to set colors and animations ofer CAN via java programs and an iphone program
